Angular2 Color Picker Directive

A Color Picker Directive for Angular 2 with no dependencies.

by Alberto Pujante

View on Github

Angular2 Color Picker

Usage:

<input [(colorPicker)]="color" 
       [style.background]="color"/>

Show text input color:

<input [(colorPicker)]="color" 
       [style.background]="color" 
       [value]="color"/>


Output format:

<input [(colorPicker)]="color" 
       [style.background]="color" 
       [value]="color"
       [cpOutputFormat]="'rgba'"/>

Changing background color and input value:

<input [(colorPicker)]="color" 
       [style.background]="color" 
       [value]="color"
       [cpPosition]="'bottom'"/>
Change me!

Use it with others html elements:

<span [(colorPicker)]="color" 
      [cpPosition]="'bottom'"
      [style.color]="color">Change me!</span>

Options Values (default values in bold)
cpOutputFormat 'hex', 'rgba', 'hsla'
cpPosition 'right', 'left', 'top', 'bottom'
